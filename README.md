## Setup Instructions

### Clone Repository

git clone <https://github.com/lazycoder54/rest-api-project.git>

cd project

### Configure Environment Variables

Create .env file:

GROQ_API_KEY=your_key_here

### Run Application

docker compose up --build

### Services

Frontend:
http://localhost:3000

Backend:
http://localhost:5000

AI Service:
http://localhost:8000/docs
